# 设计文档模板

复制新文档时，先填写以下 YAML frontmatter：

```yaml
---
title: 待命名设计文档
decision_status: 待探索
validation_status: 未验证
domains: []
architecture_layers: []
related_specs: []
related_plans: []
updated_at: 2026-03-27
---
```

状态枚举、字段含义和同步规则以 `docs/DESIGN.md` 为准。

### 待探索 / 比较中

写出当前阶段的决策快照，至少包含：

- 已排除的方案及排除依据
- 仍在候选的方案各自的阻塞条件——满足什么条件即可选定、或发现什么事实即应排除
- 下一步需要执行的验证动作（具体到可执行的命令、实验或代码路径检查）

不得将未经验证的判断写成已接受结论。
